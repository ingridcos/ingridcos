## 📌 Dashboard de RH no BI - Hashtag Treinamentos

Projeto de construção de um **Dashboard de RH**, simulando o dia a dia de um profissional de **People Analytics** dentro de uma empresa.

Desenvolvido em conjunto com a equipe da Hashtag Treinamentos, com foco em análise de dados aplicada à área de Recursos Humanos.

### 📊 Funcionalidades do Dashboard

O dashboard apresenta:

- Distribuição de funcionários por cidade  
- Distribuição de funcionários por gênero  
- Árvore hierárquica de funcionários ativos por cargos  
- Tooltip interativo nos gráficos de funcionários por cidade, exibindo horas extras por ocupação  

Repositório do projeto
https://github.com/ingridcos/Dashboards-de-RH-no-BI
